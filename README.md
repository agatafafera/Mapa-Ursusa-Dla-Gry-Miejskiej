# Mapa Ursusa - Tory Do Wolności

Artystyczna mapa dzielnicy Ursus przygotowana na potrzeby promocji gry miejskiej Tory do Wolności w ramach projektu Kod Ursusa 1976-2026, podczas moich praktyk studenckich w Fundacji Polskie Pałace Na Źródle. 

Prosty kod generujący mapę w stylu lat 70 za pomocą biblioteki prettymaps i map OpenStreetMap. 

## Oświaczenie o użyciu AI: 
Wykorzystanie Claude Desktop w roli tutora: pomoc w przejściu przez konfigurację środowiska, wytłumaczenie fragmentów kodu, dobranie kodów kolorów pod wybraną stylistykę, nauka używania Gita. 
Kod nie jest wygenerowany.
